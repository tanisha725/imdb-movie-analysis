#  IMDb Movie Analysis & Rating Prediction

##  Project Overview

This project focuses on analyzing IMDb movie data to uncover patterns, trends, and factors that influence movie ratings. Through Exploratory Data Analysis (EDA), data visualization, and Machine Learning techniques, the project provides meaningful insights into audience preferences and movie performance metrics.

The objective is to understand how attributes such as runtime, votes, metascore, and release year impact IMDb ratings and to build predictive models capable of estimating movie ratings based on these features.

---

##  Objectives

- Perform comprehensive Exploratory Data Analysis (EDA) on IMDb movie data.
- Identify key factors influencing movie ratings.
- Visualize trends and relationships among movie attributes.
- Build Machine Learning models for IMDb rating prediction.
- Compare model performance and evaluate prediction accuracy.

---

##  Dataset Features

The dataset contains movie-related attributes including:

- IMDb Rating
- Number of Votes
- Runtime (Minutes)
- Metascore
- Release Year
- Genre Information
- Movie Popularity Metrics

---

## 🔍 Exploratory Data Analysis

The analysis includes:

### Data Cleaning
- Handling missing values
- Data preprocessing
- Feature selection

### Visualizations
- Rating Distribution
- Runtime Distribution
- Votes Analysis
- Metascore Analysis
- Correlation Heatmaps
- Feature Relationship Plots
- Outlier Detection

### Statistical Insights
- Impact of popularity on ratings
- Relationship between critic scores and audience ratings
- Trends across release years

---

##  Machine Learning Models

The following regression models were implemented and evaluated:

- Decision Tree Regressor
- Random Forest Regressor
- K-Nearest Neighbors (KNN)
- XGBoost Regressor

These models were trained to predict IMDb ratings using movie metadata and performance indicators.

## Key Insights

- Movies with higher vote counts generally receive more stable ratings.
- Metascore shows a positive correlation with IMDb ratings.
- Popularity metrics significantly influence audience ratings.
- Ensemble learning models outperform basic regression approaches.
- Feature engineering improves prediction performance.
